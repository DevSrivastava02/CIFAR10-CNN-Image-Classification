CIFAR10-CNN-Streamlit-App


This project implements a Convolutional Neural Network (CNN) using TensorFlow and Keras to classify images from the CIFAR-10 dataset. The trained model is deployed using a Streamlit web application for real-time image prediction.

Tech Stack

Python

TensorFlow / Keras

Streamlit

NumPy, PIL

#### Run Locally
python -m streamlit run app.py
